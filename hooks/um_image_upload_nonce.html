Filter: um_image_upload_nonce

apply_filters( 'um_image_upload_nonce', $verify_nonce ) → {bool}

Filters image upload checking nonce.

Parameters:
Name Type Description
$verify_nonce bool

Verify nonce marker. Default true.
Since:
  • 1.3.x
Source:
  • includes/core/class-files.php, line 484
Returns:

Verify nonce marker.

Type
bool
Example

Disable checking nonce on image upload.

function my_image_upload_nonce( $verify_nonce ) {
    // your code here
    $verify_nonce = false;
    return $verify_nonce;
}
add_filter( 'um_image_upload_nonce', 'my_image_upload_nonce' );
